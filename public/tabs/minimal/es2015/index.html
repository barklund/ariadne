What is this?

This is an example implementation of ARIA best practices for a minimal tab widget in Vanilla JS (ES2015).

As a part of the Ariadne project, this aims to inspire developers to consider also accessibility aspects of their web applications and in particular see, how little it takes to implement something that everyone can use regardless of abilities.
